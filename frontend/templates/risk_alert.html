Real-Time Risk — High-Risk Alerts
The High-Risk Alert module classifies time periods as high-risk or low-risk using logistic regression modeling. It evaluates prediction accuracy using ROC curve analysis. This enables early-warning alerts for potential crime escalation.

ROC Curve — Model Accuracy

The ROC curve evaluates classification performance by plotting true positive rate against false positive rate. A higher AUC score indicates stronger predictive capability. This helps validate alert prediction reliability.

High vs Low Risk Distribution

This donut chart shows the proportion of high-risk and low-risk periods detected by the model. It provides a quick overview of overall risk segmentation and alert frequency.
